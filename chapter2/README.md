# Chapter 2: 자바 기본 문법과 데이터 타입

이 챕터에서는 자바 프로그래밍 언어의 기본 문법과 데이터 타입에 대해 학습합니다.

## 학습 목표
- 자바의 기본 데이터 타입(primitive types) 이해하기
- 변수 선언 및 초기화 방법 학습하기
- 연산자(산술, 비교, 논리, 대입 연산자 등) 사용법 익히기
- 문자열(String) 다루기
- 형변환(Type Casting) 이해하기
- 상수(Constants) 선언 및 사용법 익히기
- 주석(Comments) 작성 방법 배우기

## 예제 코드
- `ex1`: 기본 데이터 타입과 변수 사용 예제
- `ex2`: 연산자 사용 예제
- `ex3`: 문자열 처리 예제
- `ex4`: 형변환 예제

## 실습 과제

### 실습 1: 변수와 데이터 타입
**목표**: 다양한 자바 데이터 타입을 이해하고 변수를 선언하고 초기화합니다.

**요구사항**:
1. 모든 기본 데이터 타입(byte, short, int, long, float, double, char, boolean)을 사용하여 변수를 선언하고 초기화하세요.
2. 각 변수의 값을 출력하세요.
3. 리터럴 표기법의 다양한 형태를 사용해보세요(정수 리터럴, 실수 리터럴, 문자 리터럴, boolean 리터럴).

**참고 사항**:
- 정수 리터럴: `123`, `0xFF`(16진수), `0b1010`(2진수), `076`(8진수)
- 실수 리터럴: `3.14`, `2.0e3`(지수 표기법)
- long 타입에는 `L` 접미사, float 타입에는 `F` 접미사를 사용하세요.

### 실습 2: 연산자와 표현식
**목표**: 다양한 연산자를 사용하여 계산을 수행합니다.

**요구사항**:
1. 산술 연산자(`+`, `-`, `*`, `/`, `%`)를 사용하여 간단한 계산기 프로그램을 만드세요.
2. 증감 연산자(`++`, `--`)의 전위와 후위 형식 차이를 보여주는 코드를 작성하세요.
3. 비교 연산자와 논리 연산자를 사용하여 조건식을 만드세요.
4. 비트 연산자(`&`, `|`, `^`, `~`, `<<`, `>>`, `>>>`)를 사용한 예제를 작성하세요.

**참고 사항**:
- 정수 나눗셈(`/`)은 소수점 이하를 버린 결과를 줍니다.
- 변수 값을 변경하는 연산자(`=`, `+=`, `-=`, `*=`, `/=`, `%=`)를 활용해보세요.

### 실습 3: 문자열 다루기
**목표**: 문자열 연산과 메소드를 활용합니다.

**요구사항**:
1. 여러 문자열을 연결하는 코드를 작성하세요.
2. 다양한 String 메소드(`length()`, `charAt()`, `substring()`, `indexOf()`, `toUpperCase()`, `toLowerCase()`, `trim()`, `replace()` 등)를 사용해보세요.
3. String과 기본 타입 간의 변환 메소드를 사용해보세요.

**참고 사항**:
- String은 불변(immutable) 객체입니다.
- String 연결 연산자 `+`를 사용하면 새로운 String 객체가 생성됩니다.
- 문자열과 숫자를 더하면 자동으로 문자열로 변환됩니다.

### 실습 4: 형변환 실습
**목표**: 암시적(자동) 형변환과 명시적(강제) 형변환의 차이를 이해합니다.

**요구사항**:
1. 자동 형변환이 발생하는 예제를 작성하세요.
2. 명시적 형변환을 사용해야 하는 경우의 예제를 작성하세요.
3. 형변환 과정에서 발생할 수 있는 데이터 손실 사례를 보여주세요.

**참고 사항**:
- 작은 타입에서 큰 타입으로의 변환은 자동으로 이루어집니다.
- 큰 타입에서 작은 타입으로의 변환은 명시적 캐스팅이 필요합니다: `(타입)값`
